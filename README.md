# 🚲 Bike Sharing Analysis Dashboard

## ℹ️ About

Proyek ini merupakan analisis data pada **Bike Sharing Dataset** untuk mengeksplorasi tren penyewaan sepeda berdasarkan parameter waktu, kondisi cuaca, dan profil hari. Dashboard interaktif ini dibuat untuk memberikan insight mendalam bagi pengelola layanan penyewaan sepeda.

## 🚀 Features

- **Time Trend Analysis:** Visualisasi pertumbuhan penyewaan dari 2011 hingga 2012.
- **Hourly Peak Monitoring:** Identifikasi jam operasional tersibuk.
- **Weather Impact Insight:** Analisis korelasi kondisi cuaca terhadap volume penyewaan.
- **Interactive Filtering:** Filter data berdasarkan rentang tanggal dan jenis cuaca secara real-time.

## 🛠️ Technologies Used

- **Language:** Python 🐍
- **Analysis:** Jupyter Notebook (VS Code)
- **Libraries:** Pandas, Matplotlib, Seaborn
- **Dashboard:** Streamlit 🎈

## ⚙️ Setup Instructions

### 1. Persiapan Folder

Pastikan struktur folder kamu sudah sesuai:

- `dashboard/` (isi `dashboard.py` & `main_data.csv`)
- `data/` (isi `day.csv` & `hour.csv`)

### 2. Install Dependencies

Buka terminal di VS Code dan jalankan:

```bash
pip install -r requirements.txt
```
